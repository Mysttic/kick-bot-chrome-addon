# Privacy Policy - Kick.com Chat Monitor

**Last updated:** January 2026

## Overview

Kick.com Chat Monitor is a browser extension that monitors chat messages on kick.com and triggers automated actions based on user-defined keywords. This privacy policy explains how we handle user data.

## Data Collection

**We do NOT collect any personal data.**

This extension:
- ❌ Does NOT collect personally identifiable information
- ❌ Does NOT collect browsing history
- ❌ Does NOT collect financial information
- ❌ Does NOT track user location
- ❌ Does NOT transmit any data to external servers

## Data Storage

All configuration data (triggers, keywords, actions) is stored **locally** on your device using Chrome's built-in `chrome.storage.local` API. This data:
- Never leaves your browser
- Is not synced to any cloud service
- Is not accessible by third parties
- Is deleted when you uninstall the extension

## Permissions Used

| Permission | Purpose |
|------------|---------|
| `storage` | Save your trigger configurations locally |
| `notifications` | Display desktop notifications when triggers match |
| `scripting` | Inject content script to monitor kick.com chat |
| `host_permissions` (kick.com) | Access kick.com pages to read and interact with chat |

## Third-Party Services

This extension does **NOT** use any third-party services, analytics, or tracking tools.

## Data Sharing

We do **NOT** sell, trade, or transfer your data to any third parties.

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date above.

If you have questions about this privacy policy, please open an issue on our GitHub repository.
